Every HTML element can be displayed in one of three different ways: block, inline, or inline-block. These properties affect how content interacts with other content surrounding it. This is a summary of the default styles. Of course, these styles can be applied to any element and the default overridden.

Display: Inline;

Inline objects are things that follow the flow of text without breaking it. That is to say that it will only take up as much width as needed.

Tags that have this as their default style include a, li, span, img, and styling tags like strong and em.

You'll notice the code elements of this presentation have different styles applied to them. The code tag is another example of an inline element.

Display: Block;

Block elements both start and force new lines. Block elements do not tolerate anything beside them, and the outer edges of these elements will touch the edges of the parent containers.

The default display: block ; value is set for p tags, and h1-h6 tags.

div tags also force new lines. All of the figures in this presentation are made using div tags, and the default styles have been overridden to demonstrate different display properties.

Display: Inline-Block;

Setting inline-block allows the element to stay inline while preserving the ability to control width, height, margin and padding parameters of different elements, which could all be on the same line.
